# Workforce-Wellness-Analytics
End-to-end Workforce Wellness Analytics Dashboard using SQL Server and Power BI and Excel


## Project Overview

This project presents an end-to-end Business Intelligence solution focused on workforce wellness, behavioral analytics, and employee wellbeing.

The solution combines:

* Data cleaning and preprocessing
* SQL Server data modeling
* Analytical SQL views
* KPI engineering
* Interactive Power BI dashboards

The project analyzes relationships between:

* Sleep quality
* Stress levels
* Physical activity
* Sedentary behavior
* Burnout exposure
* Lifestyle patterns

---

# Tools & Technologies

* Microsoft SQL Server
* Power BI Desktop
* Microsoft Excel
* SQL
* Data Visualization
* Business Intelligence

---

# Datasets Used

The project integrates multiple public wellness and behavioral datasets containing more than 4,000 total records.

### Main Domains

* Employee burnout & HR analytics
* Sleep & stress behavior
* Physical activity tracking
* Nutrition & lifestyle behavior

---

# Data Preparation Process

## Excel Cleaning

The datasets were cleaned and standardized using Excel.

Main preprocessing steps included:

* Null value handling
* Column standardization
* Data type corrections
* Duplicate inspection
* Schema preparation

---

# SQL Server Data Modeling

The cleaned datasets were imported into SQL Server and transformed into analytical views.

### SQL Tasks Performed

* Schema organization
* Data transformation
* CASE statement categorization
* KPI logic creation
* Business segmentation
* Analytical view creation

### Main SQL Views

* vw_sleep_analysis
* vw_burnout_risk
* vw_activity_analysis
* vw_nutrition_analysis

---

# Power BI Dashboard Structure

## Page 1 — Executive Workforce Overview

High-level executive overview of:

* Workforce wellness
* Burnout exposure
* Sleep quality
* Activity behavior
* Nutrition patterns

Main insights:

* Sleep quality strongly impacts stress levels
* Sedentary behavior remains significant
* Poor nutrition patterns dominate a large portion of the population

---

## Page 2 — Sleep & Wellness Analytics

Focused analysis on:

* Sleep duration
* Stress relationships
* Sleep segmentation
* Occupational sleep behavior

Main insights:

* Lower sleep duration correlates with higher stress
* Healthy sleep behavior is associated with increased activity levels
* Sleep quality varies across occupations

---

## Page 3 — Activity & Lifestyle Analytics

Focused analysis on:

* Activity intensity
* Sedentary behavior
* Calories burned
* Lifestyle segmentation

Main insights:

* Highly active individuals burn significantly more calories
* Sedentary groups accumulate substantially higher inactive time
* Strong separation exists between active and inactive behavioral groups

---

# Key Business Insights

* Employee wellness is strongly connected to behavioral patterns.
* Poor sleep quality is associated with elevated stress levels.
* Sedentary behavior significantly impacts lifestyle quality.
* Burnout exposure increases when work-life balance decreases.
* Physical activity positively influences overall wellness indicators.
